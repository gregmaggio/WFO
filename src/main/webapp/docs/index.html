Getting Started

To begin using the WFO HTTP API, follow the steps below.

You can enter http://datamagic.ca/WFO/list in your browser to see all WFO.

You can enter http://datamagic.ca/WFO/LWX in your browser to see the WFO information for Baltimore MD/Washington DC.

You can enter http://datamagic.ca/WFO/39.006733/-76.917968/coordinates in your browser to see the WFO information for Baltimore MD/Washington DC. This method returns an array of WFO in case there are multiple matches.
